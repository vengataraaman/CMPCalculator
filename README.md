This Calculator app is a modern Compose Multiplatform app written in ![Kotlin](https://img.shields.io/badge/Kotlin-7F52FF?style=flat-square&logo=kotlin&logoColor=white). The app provides standard functionality, basic arithmetic operations such as addition, subtraction, multiplication, and division.
